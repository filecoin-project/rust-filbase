# Filbase

> Filecoin proofs & sector management in a convenient package.


**Warning**: Requires a _new_ rust nightly.


## Building

```sh
> cargo build --release
```

## Usage

```sh
# Start the daemon
> filbase daemon

# In another terminal
> filbase sector size
  1024
```


## Testing

```sh
> cargo test
```
